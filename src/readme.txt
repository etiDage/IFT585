Dans Linux, executer le script buildHost.sh pour build les containers du sender et receiver. 

Pour LS:
Ensuite executer le script buildRouteursLS.sh et ensuite runRouteurLS.sh. Faire un docker run -it receiver et un docker run -it sender dans 2 autres terminaux.

Pour DV:
Ensuite executer le script buildRouteurDV.sh et ensuite runRouteurDV.sh. Faire un docker run -it receiver et un docker run -it sender dans 2 autres terminaux.
